Place your own photos and videos in this folder.

Use the filenames listed in the main `README.md`, for example:

- `cover-hero.jpg`
- `collection-01.webp`
- `factory-film.mp4`
- `factory-process-flow.png`
- `product-01-front.jpeg`
- `product-01-back.jpeg`

The website automatically checks this folder and loads the matching files into the correct lookbook slots.

For product boards, keep the number matched to the product order:

- Product 01 uses `product-01-front` and `product-01-back`.
- Product 02 uses `product-02-front` and `product-02-back`.
- Product 03 uses `product-03-front` and `product-03-back`.

You can continue the same pattern for as many product boards as needed.
